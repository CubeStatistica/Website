Course Content

  • Week_1

    Introduction to Software Stack & Purpose of the Course. (Installing R, Jira/GitHub Projects, Folder Structuring, GitHub Introduction, Scrum & Dashboard knowledge) - We help everyone make a GitHub, Slack account.

  • Week_2

    R Nuts & Bolts (Basic R commands, R Markdown (+ some script), Introduction to Tidy verse Package, Getting and Cleaning Data) & Flex dashboard in Action (Creating & Deploying Dashboards, very basic custom stylizing using CSS). Present sample dashboard to class to set their goal for what they should achieve or exceed).

  • Week_3

    Exploratory Data Analysis (Dealing with Missing Values, Outlier Detection, ggplot Visualizations) Detailed Getting and Cleaning Data (Manipulating, Grouping and Cleaning Data in R) + Application of this in dashboard.

  • Week_4

    Basic Machine Learning Concepts & Intro to R shiny (Train, Valid, Test concepts, data science pipeline + Visualizing and summarizing results) + Application of this in dashboard.

  • Week_5

    Dashboard in Action Part 1- Teaching more advanced App and Dashboard skills and features (shiny modules, custom stylizing using CSS) + Time for students to work on their dashboard. Making sure they have the workflow done for software development (Git bash, GitHub, Pull Request, Command Line).

  • Week_6

    Dashboard in Action Part 2 - Teaching more advanced App and Dashboard skills and features (more custom stylizing using css) + Time for students to work on their dashboard. Students showcase their projects.
